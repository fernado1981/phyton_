El ejercicio consiste en dada una api conpuesta por un diccionario y dentro otros 3 diccionarios,
escoger el primer diccionario ('parkings') y sobre el buscar las matriculas por un color dado y
crear otro diccionarioque muestre matricula{color:{vehiculos:de esa matricula con ese color}.

Ej:
3869HTV {'yellow': {'vehículos': 1}}
7720TRA {'yellow': {'vehículos': 2}}
